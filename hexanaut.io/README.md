# Hexanaut.io Zoom-out hack & Gamepad support
Hexanaut.io is a paper-io type competetive io game. It's actually really fun, i recommend playing it, especially on the Pro FFA game mode.
This user script allows you to zoom out by scrolling mouse wheel, and also playing the game with a gamepad. 

# How to install
First install Violentmonkey browser extension
- If you are using google chrome [click here](https://chromewebstore.google.com/detail/violentmonkey/jinjaccalgkegednnccohejagnlnfdag).
- If you are using firefox [click here](https://addons.mozilla.org/en-US/firefox/addon/violentmonkey/).
- If you are using microsoft edge [click here](https://microsoftedge.microsoft.com/addons/detail/violentmonkey/eeagobfjdenkkddmbclomhiblgggliao).

After you install the extension, install my script by clicking [HERE](https://github.com/KebabLord/js-hacks/raw/refs/heads/main/hexanaut.io/hexanaut.user.js).
